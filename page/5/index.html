/etc/passwd 和 /etc/group

UDI 和 GID

1, 核实是否存在账号（查看文件 passwd,group ,有的话读取 UID 和 GID；并且读取住文件夹和 shell 设置）

2, 核对密码（/etc/shadow 找到对应的账号与 UID,核对密码）
3， Shell

/etc/passwd 与 /etc/shadow 必须要让系统读取。

  • UID 系统管理员（0—） 、系统账号（1 ~ 999）、可登录账号（1000 ~ 60000）
